Nauka terminala

Poznane polecenia

Tydzień trzeci

cd - wejście do katalogu
cd .. - przejście do katalogu poziom wyzej
cd ../.. - przejście do katalogu dwa poziomy wyzej
ls - lista zawartości katalogu
pwd - pokazanie ścieżki katalogu, w którym się znajdujemy
mkdir - utworzenie nowego katalogu
code . - otwórz plik w Visual Studio Code

Tydzień czwarty

git config --global user.name "" oraz git config --global user.email - ustawienia w git
git config --get user.name "" oraz git config --get user.email - sprawdzenie ustawień
git init - tworzy nowe puste repozytorium w katalogu
git status - podgląd plików w katalogu i ich stan
git add . - dodanie plików do repozytorium
git commit -m "nazwa zmian"
gitk - pokazuje historię zmian w wersji graficznej
git push - komenda wrzucenia zmiany na github
Podstawowy flow poleceń w terminalu: git add . / git commit -m "opis zmiany" / git push
