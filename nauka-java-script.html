Tydzień czwarty

Zmienne w Java Script (zmienna to wskazanie na jakąś wartość):
const - zmienne mające stałe połączenie z jedną wartością, np.:
const name = 'Patrycja';
const name = 34;
let - zmienne, których połączenie możemy przepinać, ale definiujemy tylko raz, np.:
let isHappy = false;
isHappy = true;
isHappy = 'tak';
Typy proste w JS (primitives) (określają, jakiego rodzaju dane są przechowywane w zmiennych):
String - tekst, otaczamy ' lub "
Number - liczba
Bigint - duża liczba, większa niż 253
boolean - true lub false
null - wartość celowa niezdefiniowana
undefined - brak definicji
*symbol - tworzy "unikalne opakowanie"
DOM - document object model służy do odwoływania się do elementów
document.querySelector('.selector')
nazwaZmiennejQuerySelector.innerHTML='treść' - służy do podmieniania np. treści
